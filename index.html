grammophon by lennart

Let's hear that song again, but this time we'll use the grammophon

A frontend for playdar and eqlizr

Dependencies

A lot, flex 4 sdk, yajl for as3, as3playdar, as3httpclient for flex, the as3corelib, robotlegs, swiftsuspenders a running playdar instance with blipfm resolver and the eqlizr with apache solr and still a lot of effort

Install

still alpha, currently built with project-sprouts
